CLAS12-Analysis documentation
=============================

**CLAS12-Analysis** is a groovy project for identifying all unique final state particle combinations
and computing event level kinematics.  It reads events from HIPO files and saves the selected particle
combinations to `ROOT <https://root.cern.ch>`_ TTrees.

.. note::

   This project is under active development.
